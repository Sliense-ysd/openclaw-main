## 管理命令

### 查看所有任务
```bash
openclaw cron list
```

### 查看任务详情
```bash
openclaw cron status <job-id>
```

### 手动运行任务
```bash
openclaw cron run <job-id>
```

### 查看运行历史
```bash
openclaw cron runs --id <job-id> --limit 10
```

### 禁用/启用任务
```bash
openclaw cron edit <job-id> --enabled false
openclaw cron edit <job-id> --enabled true
```

### 删除任务
```bash
openclaw cron remove <job-id>
```

## 任务配置说明

### 执行模式
- **isolated**: 独立会话执行，不影响主会话历史
- **main**: 在主会话中执行（未使用）

### 唤醒模式
- **now**: 立即触发 heartbeat（默认）
- **next-heartbeat**: 等待下次 heartbeat

### 发送模式
- **announce**: 直接发送到指定频道
- **webhook**: 发送到 webhook URL
- **none**: 不发送，仅内部执行

## 注意事项

1. **时区设置**: 所有任务使用 `Asia/Shanghai` 时区
2. **任务隔离**: 使用 isolated 模式，不会污染主会话历史
3. **Agent 绑定**: 每个任务绑定到特定的 agent，确保专业化
4. **共享记忆**: 所有任务都会读取 `~/.openclaw/team-memory/`
5. **自动重试**: 失败任务会自动重试（指数退避）
